# 🎲 Statistics & Probability — From Scratch

This repository documents my deep-dive into **Statistics & Probability**, built with a scratch-first approach — implementing every concept from the ground up in Python before relying on libraries, so the underlying intuition and math are fully understood.

Part of my larger [Data Science & ML Journey](https://github.com/Itz-Me-Sumit/Data-Science).

---

# 🧠 What's Inside

### 📊 Descriptive Statistics
* Measures of Central Tendency (Mean, Median, Mode)
* Measures of Dispersion (Variance, Standard Deviation, Range, IQR)
* Skewness & Kurtosis
* Summary statistics from scratch

### 🎯 Probability Distributions (Built from Scratch)
* **Discrete Distributions** — Bernoulli, Binomial, Poisson
* **Continuous Distributions** — Uniform, Exponential, Normal
* Each distribution implemented as a **Python class** — PDF, CDF, mean, variance, sampling — built progressively
* Focus on intuition: what the distribution models, when it applies, why the math works the way it does

### 🔢 Categorical Encoding
* Handling categorical variables statistically
* Encoding techniques and when to use which

### 🧪 Inferential Statistics
* Hypothesis Testing
* T-test, Mann-Whitney U Test, ANOVA, Chi-Square Test
* Understanding p-values, null/alternate hypotheses, test selection logic

---

# ⚙️ Approach

✔️ Scratch-first implementation (no `scipy.stats` shortcuts until the concept is internalized)
✔️ Intuition before formulas — understanding *why* before *how*
✔️ Every distribution/test built as reusable, testable Python code
✔️ Connecting theory to real datasets
